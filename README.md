# NoSafeW Media Kit

本仓库保存适合传播的图文、短视频和发帖草稿。目标用户是中文用户：不熟悉技术报告，但可能正在被 SafeW / SafeX 推广内容误导的人。

## 内容

| 路径 | 内容 |
| --- | --- |
| `images/` | 分享图、横幅图 |
| `video/final.mp4` | 成片视频 |
| `video/preview_silent.mp4` | 静音预览视频 |
| `video/frames/` | 8 张分镜帧 |
| `telegram-posts.md` | Telegram 频道 / 群转发文案 |
| `x-thread.md` | X 中文线程草稿 |
| `reddit-post.md` | Reddit / 海外论坛发帖草稿 |
| `blog-post.md` | Medium / WordPress / Blogger 中文长文草稿 |
| `video-script.md` | 短视频脚本 |
| `assets.md` | 素材清单 |

## 发帖原则

- 不刷屏。
- 不复制粘贴同一篇到几十个地方。
- 每个平台写不同角度。
- 标题面向真实问题，例如“SafeW 安全吗？”。
- 先给结论，再给证据，再给自救步骤。
