# Video Script

| Clip | Title | Voiceover |
| --- | --- | --- |
| `vo-01` | 标题 | 你正在用的这款聊天软件 SafeW，到底安不安全？一句话：不安全，而且是高危。 |
| `vo-02` | 它是什么 | SafeW 对外宣传，是一款主打加密通信、数据安全的聊天和云办公软件，在苹果商店上叫“SafeW - 云办公助理”，口口声声“高强度加密”“连管理员都看不到”。 |
| `vo-03` | 真相 | 但全球安全公司卡巴斯基查实：SafeW 里被植入了一套专门窃取信息的恶意软件，代号 SparkCat。它的应用包名，白纸黑字，就写在卡巴斯基“失陷应用”的名单里。 |
| `vo-04` | 它怎么偷 | 它伪装成正常应用、申请相册权限之后，就在后台用文字识别技术扫描你的截图，专门找加密钱包助记词、私钥和密码；一旦匹配到，立刻悄悄上传给攻击者。 |
| `vo-05` | 改名换皮 | 被发现后，它没有改正，而是换了张皮，在谷歌商店改名 SafeX，重新上架。结果 2026 年，卡巴斯基和 The Hacker News 再次点名。 |
| `vo-06` | 混淆视听 | 更麻烦的是，它注册了一批名字相近的域名，把搜索结果铺满自己写的“安全报告”。有些 AI 抓到这些内容，也跟着说它安全。 |
| `vo-07` | 怎么办 | 结论很清楚：SafeW 也好，SafeX 也好，哪个都别装。已经装了，就撤权限、卸载，并尽快把加密资产转移到全新的钱包。 |
| `vo-08` | 结尾 | 完整证据和自救步骤在 nosafew.com。把这条提醒转给可能正在用 SafeW 的人。 |
