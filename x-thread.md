# X Thread

## 中文线程

1/  
SafeW / SafeX 不建议安装，尤其是持有加密资产、相册里保存过助记词或密码截图的人。

这不是单纯“权限多一点”的问题，而是公开安全报告点名的窃密风险。

2/  
Kaspersky 的公开报告中，SafeW 相关包名 / Bundle ID 出现在受影响应用列表：

- `org.safew.messenger`
- `org.safew.messenger.store`
- `com.safew.messenger`

证据： https://nosafew.com/evidence/

3/  
The Hacker News 2026 年报道的新一轮活动中，也列出 SafeX 和 iOS 上的 SafeW - 云办公助理。

SafeW / SafeX 的问题不是“用户下错假版本”这么简单。

4/  
风险点：应用拿到图库权限后，可用 OCR 扫描相册截图，寻找钱包恢复短语、助记词、私钥、密码等内容。

如果命中，资产可能被转走。

5/  
已经装过怎么办：

- 撤销权限并卸载
- 检查相册敏感截图
- 存过助记词 / 私钥就转移到新钱包
- 修改相关密码
- 不要再把助记词截图放相册

自救： https://nosafew.com/remove/

6/  
给中文用户的提醒：不要被“安全聊天”“加密通信”“小飞机替代品”这些词带偏。

真正要看的是：客户端是否可信、是否有独立安全审计、是否被安全厂商公开点名过。
